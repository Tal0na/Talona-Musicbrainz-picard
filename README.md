## Overview

This setup is tailored for advanced classical tagging and metadata processing in MusicBrainz Picard.

Key characteristics:

- Classical work structure and artist credits are prioritized
- Cover art provider priority is documented in [`Cover/README.md`](Cover/README.md)
- Plugin usage and behavior are documented in [`Plugins/README.md`](Plugins/README.md)
- Metadata and work/tag mapping rules are documented in [`Metadata/README.md`](Metadata/README.md)

## How to Use

1. Clone this repository
2. Copy `Picard_Talona_Backup.ini` to your Picard config directory
3. Install the listed plugins via **Picard → Options → Plugins**

### Config file location by OS

| OS | Path |
|---|---|
| Windows | `%APPDATA%\MusicBrainz\Picard\` |
| Linux | `~/.config/MusicBrainz/Picard/` |
| macOS | `~/Library/Preferences/MusicBrainz/` |
